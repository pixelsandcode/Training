you can use nav-justified class for having the tabs or pills take the whole width of their parent

you can usenav-stacked class to have tabs or pills stack on eachother

you can add disabled class to <li> tag to disable the tab or pill

you can use nabvar-form class to use form in navbar

you can use navbar-btn class to use button in navbar

you can use nabvar-text class to use text in navbar

you can use nabvar-left class to make navbar apear in the left or use pull-left as mixin

you can use nabvar-right class to make navbar apear in the right or use pull-right as mixin

you can use navbar-fixed-top or navbar-fixed-bottom to fix navbar in the position you like

if you want that your navbar scrolls away with page use navbar-static-top class

use navbar-inverse class to make your navbar black
